# Research Artifacts

This directory is reserved for reviewable research outputs that should be easy for humans to inspect and discuss.

Expected future contents:

- memo templates
- postmortem templates
- sample research packets
- approved paper-trade review bundles

Raw source data should not live here. This directory is for human-facing artifacts, not ingestion payloads.

Repository-level layout conventions:

- `research_artifacts/evidence/` for reviewable evidence packs or citation bundles
- `research_artifacts/hypotheses/` for packaged thesis and counter-thesis artifacts
- `research_artifacts/features/` for human-inspectable feature definition packs
- `research_artifacts/signals/` for explainable signal summaries
- `research_artifacts/portfolio_proposals/` for proposal review bundles
- `research_artifacts/paper_trades/` for operator-facing paper trade packets
- `research_artifacts/memos/` for memo templates and generated memo bundles
- `research_artifacts/audit_logs/` for human-facing audit exports, not the system of record
